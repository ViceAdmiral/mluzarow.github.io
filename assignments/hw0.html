Objective
In this assignment, we will investigate the basic console IO, variable types, and formatting available in C#.
Task
Create a guided shopping experience wherein a used will choose how many of each stock item they wish to purchase. Upon iterating through each stock option, a receipt will be printed.

Note that the text formatting must be identical to the examples provided
Input Format
The user will be queried during runtime as to how many of each item they wish to purchase. The input will be in the form of an integer. At this point, you do not have to check the user input, as it can be assumed to always be of type int.
Output Format
A formatted receipt detailing the amounts, costs, and total cost of all purchased items.

*********************************************

What would you like to purchase today?

Explanation
We print out the static header, detailing the prices off all items. Then, in a predefined order, each item is printed to the console suffixed with a question mark. Immediately after each item query, the user will enter an integer denoting the count of that item they wish to purchase. After an entry is made for each item, the receipt is printed.

Note that when calculating tax, always trim trailing numbers beyond the hundredths place.

Note that you do not have to worry about price overflow beyond $9,999.99.

Note that the output structure and number formatting must match the example perfectly.
